# 🌍 Air Pollution Analysis and Air Quality Prediction

## 📌 Project Overview
This project focuses on analyzing air pollution data and predicting air quality levels using Machine Learning techniques. It explores environmental factors such as PM2.5, PM10, NO2, SO2, CO, temperature, and humidity to understand their impact on air quality.

The project also includes data visualization and classification modeling to predict air quality categories such as **Good, Moderate, Poor, and Hazardous**.

---

## 🎯 Objectives
- To analyze air pollution data using exploratory data analysis (EDA)
- To understand the relationship between environmental factors and air quality
- To build a machine learning model for air quality prediction
- To visualize key insights using graphs
- To improve data-driven decision making for environmental monitoring

---

## 📊 Dataset Information
- **Total Records:** 5000  
- **Total Features:** 10  
- **Target Variable:** Air Quality  

### Features:
- Temperature  
- Humidity  
- PM2.5  
- PM10  
- NO2  
- SO2  
- CO  
- Proximity to Industrial Areas  
- Population Density  

### Target Classes:
- Good  
- Moderate  
- Poor  
- Hazardous

## 📈 Project Workflow
1. Data Collection  
2. Data Cleaning  
   - Inspection for missing values and duplicacy
   - Verification of data types.
   - label encoding
3. Exploratory Data Analysis (EDA)
   - Overview
   - Data visualization  
   - Pattern identification  
4. Feature Selection  
5. Model Training  
   - Machine Learning classification model  
6. Model Evaluation  
7. Prediction of Air Quality   

---

## 📊 Key Insights
- PM2.5 and PM10 significantly impact air quality  
- Higher pollution levels are linked with industrial proximity  
- Population density affects pollution concentration  
- Machine learning models can effectively classify air quality levels
